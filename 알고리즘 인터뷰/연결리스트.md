# 연결리스트
> 연결리스트는 데이터 요소의 선형집합으로, 데이터의 순서가 메모리에 물리적인 순서대로 저장되지 않는 것

## 연결리스트의 시간 복잡도 
연결리스트는 배열과 달리 특정 인덱스에 접근 할 때 `전체 순서`를 읽어야함
그러므로 상수시간이 걸릴 수가 없으며 탐색을 할때에 `O(n)`이 소요됨

* 연결리스트 탐색 시간 : O(n)
* 연결리스트 시작 지점에 추가 : O(1)
* 연결리스트 시작 지점에 삭제 : O(1)
* 연결리스트 시작 지점에 추출 : O(1)
  
___
* 연결리스트 끝 지점에 추가 : O(1)
* 연결리스트 끝 지점에 삭제 : O(1)
* 연결리스트 끝 지점에 추출 : O(1)
